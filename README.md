# API - Events

## How to install and run on terminal


#### It is necessary to install node: https://nodejs.org/en/download/

&nbsp;

### - Clone repository
```
  git clone https://github.com/FranciscaFigueiredo/API-events.git
```
&nbsp;

### - Install dependencies

```
  npm i
```
or
```
  npm install
```

&nbsp;

### - Start

```
  npm run start:dev
```
